## While 반복문
- While 조건 : <br> 
반복할 코드
조건이 참인 경우에 코드를 계속 반복

n = 1
while n <= 10:
    print(n)
    n += 1   # n = n + 1

### += 연산자 <br> 
- 대입 연산자의 일종 더하기 연산 후 할당
n += 1은 n = n+1이라는 의미 <br> 
-= , *=, /=, **=, //=, %=

### break
- 반복문을 즉시 종료한다.

### continue
- 특정 조건은 건너뛰고, 다음 조건 계속 실행

### 무한 반복문, 무한 루프
- 조건식에 True를 입력해 항상 참이 되도록 하여 무한히 반복하게 한다.
- 무한 반복에서 벗어나려면 ctrl + c를 누른다.
