### 비교 연산자
연산자 | 의미
------|----
\>| ~가 ~보다 크다.
\>=| ~가 ~보다 크거나 같다.
\<| ~가 ~보다 작다.
\<=| ~가 ~보다 작거나 같다.
\=| ~와 ~가 같다.
\!=| ~와 ~가 같지 않다.

### 논리 연산자
연산자 | 의미
------|----
a and b|a와 b가 모두 True일 때만 True 아니면 False
a or b|a와 b 중 하나라도 True면 True 둘다 False면 False
not a|True면 False로 False면 True로

논리식 | 결과
------|----
True and True  | True
True and False | False
False and True | False
False and False| False

논리식 | 결과
------|----
True or True  | True
True or False | True
False or True | True
False or False| False
