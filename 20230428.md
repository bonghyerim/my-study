#### http 상태 코드
http 상태 코드 | 설명
---- | ----

400 | Bad Request 요청이 잘못됨

403 | Forbidden 접근 권한 없음
404 | Not Found 요청받은 내용이 없음
405 | Method Not Allowed 접근 방법이 잘못됨
500 | Internal Server Error 서버 에러
501 | Not Implemented
502 | Bad Gateway 잘못된 응답
