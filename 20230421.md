## Python
### tuple(튜플)형
- 값이 정해지면 값을 수정할 수 없다
- 튜플은 element의 값을 수정할 수 없다.
- 수정을 하면 안 되는 데이터에 사용

### Mutable / Immutable
분류 | 자료형
--------|--------------------
Mutable | 리스트(list), 딕셔너리(dictionary)
Immutable | 숫자(number), 문자열(string), 튜플(tuple)

### zip()
- 길이가 같은 list를 묶어서 for문 등으로 사용 가능한 iterable을 반환한다.
